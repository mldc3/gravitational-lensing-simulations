# References and context

This repository is based on the student's own original project and on the accompanying course material. The new Markdown documentation was prepared from:

1. The Python scripts in the original repository.
2. The PDF notes titled *“Explicación del código e introducción teórica”*.
3. The presentation images supplied together with the project.

## Scientific context mentioned in the documentation

- A. Einstein's general-relativistic prediction that light is deflected by gravity.
- The concept of the Einstein ring produced by strong alignment.
- The first confirmed gravitational lens system: **Q0957+561** (the Twin Quasar, discovered in 1979).
- Famous and relevant lensing systems such as **JWST-ER1**, **J162746.44-005357.5**, and **SDSS J0038+4133**.
